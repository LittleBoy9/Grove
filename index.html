Features

Everything you need, without feeling overloaded.

Grove is not trying to be every developer tool at once. It focuses on the Git actions that matter every day and wraps them in a richer, faster workspace.

Multi-repo command center

See all your repositories in one place, sort dirty work to the top, create your own order, and group related projects together.

workspace scan groups drag reorder

Fast, visual Git operations

Stage, unstage, discard, diff, commit, amend, fetch, pull, push, force-push, stash, tag, and remote management all live inside one polished interface.

diffs history remotes

Launch straight into work

Open the selected repo instantly in your editor, terminal, or Finder, using your own setup instead of adapting to a closed environment.

VS Code iTerm2 Finder

Optional AI commit flow

Generate commit messages from staged diffs using Anthropic, OpenAI, or Gemini, while keeping configuration local to the app.

Developers

Built like a real desktop product, not just a styled demo.

Grove keeps a clean separation between the React frontend and the Rust-backed Tauri shell, while still using your system Git so existing credentials and hooks continue to work.

Product stack

Tauri 2 for the desktop shell, React 19 and TypeScript for the UI layer, Rust for native commands, Tailwind CSS v4 and shadcn/ui for component styling.

Git architecture

Git actions run through the system git binary, so SSH keys, credentials, hooks, and user config behave the way developers already expect.

$ git status --porcelain
$ git fetch --prune
$ git push --force-with-lease
